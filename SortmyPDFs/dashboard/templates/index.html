Starten

In der SortmyPDFs-venv installieren und starten:
  • cd /home/tim/.openclaw/workspace/SortmyPDFs
  • source .venv/bin/activate
  • pip install -r dashboard/requirements.txt
  • uvicorn dashboard.app:app --host 127.0.0.1 --port 8080
Heimnetz (LAN) + Buttons:
  • SORTMYPDFS_DASH_USER=... SORTMYPDFS_DASH_PASS=...
  • SORTMYPDFS_DASH_BUTTONS=1 uvicorn dashboard.app:app --host 0.0.0.0 --port 8080
Live-Inbox (Graph) optional:
  • SORTMYPDFS_DASH_LIVE_INBOX=1 ...
